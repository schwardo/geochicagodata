Geo Chicago Data

This application was created during the Apps for Metro Chicago Google hackathon to demonstrate the diversity of data made available by the city on the City of Chicago Data Portal. It queries all entries across each of the data sets near a particular geographical address.

This application will not be entered in the Apps for Metro Chicago contest, but feel free to use it as tool for gathering additional ideas or steal its code to use in your own project. The source code is available on GitHub at geochicagodata. The application was developed in Python to run on Google App Engine but should be easy to port it to Python environments (check the history in Git before App Engine-specific optimizations were added) or other languages (it uses JSON and not any particular Python client library for the data).
